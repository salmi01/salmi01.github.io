Epidemic Simulation

This project consists of creating an application that simulates the spread of contamination of viral origin in a city, as well as the fight against this contamination by some of these citizens, The application is coded in C language. This project aims to implement the knowledge seen in class in order to create an epidemic simulation. We use in particular the different interprocess communication mechanisms: signals, pipes, message queues, shared memory, threads and semaphores.
